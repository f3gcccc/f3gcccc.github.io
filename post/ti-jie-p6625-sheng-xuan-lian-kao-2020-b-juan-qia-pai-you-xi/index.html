题解：P6625 [省选联考 2020 B 卷] 卡牌游戏

2024-09-25

算法：前缀和+贪心

首先做一个前缀和，这是显而易见的。

然后你会发现，前缀和是负数的对答案没有用，所以答案就是 ∑i=1nai\sum\limits_{i=1}^{n}a_{i}i=1∑n​ai​

#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n;
	cin>>n;
	long long ans=0,s=0;
	for(int i=1;i<=n;i++)
	{
		int x;
		cin>>x;
		s+=x;
		if(i>1&&s>0) ans+=s;
	}
	cout<<ans<<"\n";
	return 0;
}
